## 🎯 使用流程

### 场景 1: 安排面试后发送邮件

1. HR 创建面试安排
2. 面试创建成功（**不自动发送邮件**）
3. HR 进入面试详情页
4. 点击"发送邮件给候选人"
5. 编辑邮件主题和内容
6. 点击"发送"
7. 候选人收到邮件

### 场景 2: 修改面试安排后发送邮件

1. HR 修改面试时间/地点
2. 修改保存成功（**不自动发送邮件**）
3. HR 进入面试详情页
4. 点击"发送邮件给候选人"
5. 编辑改期说明
6. 点击"发送"
7. 候选人收到改期通知

### 场景 3: 面试完成后发送结果通知

1. 面试官完成面试并提交反馈
2. HR 查看面试结果
3. HR 进入面试详情页
4. 点击"发送邮件给候选人"
5. 编辑结果通知内容
6. 点击"发送"
7. 候选人收到结果通知

## 🔧 技术细节

### 后端 API

**接口**: `POST /interviews/:id/send-candidate-email`

**请求参数**:
```json
{
  "subject": "面试通知 - 前端开发工程师",
  "content": "<div>尊敬的 XXX 您好...</div>"
}
```

**响应**:
```json
{
  "message": "邮件已成功发送"
}
```

### 前端组件

**邮件编辑弹窗**:
```tsx
interface EmailModalProps {
  show: boolean;
  interview: Interview;
  onClose: () => void;
  onSend: (subject: string, content: string) => Promise<void>;
}
```

**默认邮件模板**:
```typescript
const defaultContent = `
尊敬的 ${candidateName} 您好：

感谢您应聘我司 ${positionTitle} 职位。
经过初步筛选，我们诚挚邀请您参加面试：

面试时间：${startTime}
面试形式：${format}
面试地点：${location || meetingUrl}
面试官：${interviewerName}

请您准时参加。
祝您面试顺利！

Moka 面试系统
`;
```

## 📊 重构效果

### 改进前 ❌
- 自动发送邮件，HR 无法控制
- 模板固定，无法个性化
- 候选人收到冷冰冰的系统邮件
- 改期时无法添加说明

### 改进后 ✅
- HR 完全控制发送时机
- 每封邮件都可个性化编辑
- 候选人收到温暖的个性化邮件
- 改期时可以添加道歉和说明
- 提升候选人体验和公司形象
