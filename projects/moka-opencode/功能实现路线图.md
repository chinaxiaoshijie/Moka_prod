## 📝 备注

### 技术选型建议

- **日历库**: FullCalendar 或 react-big-calendar
- **图表库**: Recharts 或 @ant-design/charts
- **文件上传**: react-dropzone
- **Excel处理**: xlsx 或 sheetjs
- **邮件服务**: Nodemailer + SMTP

### 依赖安装

```bash
# 日历
npm install @fullcalendar/react @fullcalendar/daygrid @fullcalendar/timegrid

# 图表
npm install recharts

# 文件上传
npm install react-dropzone

# Excel
npm install xlsx
```

### 注意事项

1. 保持代码风格一致性
2. 及时更新API文档
3. 关注性能优化（大数据量列表）
4. 做好错误处理和边界情况处理
5. 保持与现有数据库模型的兼容性
