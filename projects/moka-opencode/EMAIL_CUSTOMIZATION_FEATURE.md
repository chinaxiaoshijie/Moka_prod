# HR 可编辑邮件功能说明

**功能状态**: ✅ 已实现  
**实现时间**: 2026-04-02 之前  
**验证时间**: 2026-04-02 19:30

---

## 📋 功能描述

HR 可以在发送面试通知给候选人时，**自定义编辑邮件的主题和内容**，而不是使用系统默认模板。

---

## 🎯 使用场景

### 场景 1: 特殊面试安排
HR 需要添加额外的面试说明，如：
- 携带材料清单
- 特殊注意事项
- 个性化欢迎语

### 场景 2: 重新安排面试
HR 需要解释面试变更原因，添加道歉或说明文字。

### 场景 3: 个性化沟通
HR 希望用更亲切、个性化的语言与候选人沟通。

---

## 💻 操作步骤

### 1. 进入面试详情页
```
路径：/interviews/[id]
```

### 2. 点击"发送邮件给候选人"按钮
- 位置：面试详情页右上角
- 按钮颜色：蓝色（#4371FF）

### 3. 编辑邮件内容
弹窗包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 收件人 | 只读文本 | - | 候选人邮箱地址 |
| 邮件主题 | 文本输入框 | ✅ | 自定义邮件主题 |
| 邮件内容 | 多行文本框 | ✅ | 支持 HTML 格式 |

### 4. 发送邮件
- 点击"发送邮件"按钮
- 系统显示发送状态（发送中...）
- 发送完成后显示结果（✅ 成功 / ❌ 失败）

## 🔧 技术实现

### 后端 API

**接口**: `POST /interviews/:id/send-candidate-email`

## 📝 注意事项

### 1. 必填字段
- 邮件主题：必填
- 邮件内容：必填

### 2. HTML 支持
- 邮件内容支持 HTML 格式
- 可以添加链接、图片、表格等
- 系统会自动添加签名 "Moka 面试系统"

### 3. 发送限制
- 只能发送给该面试的候选人
- 需要 HR 权限
- 邮件发送失败会显示错误信息

### 4. 最佳实践
- 保持邮件内容简洁明了
- 使用正式的商务语言
- 包含必要的面试信息
- 提供联系方式以便候选人咨询
