## 核心工作流程

### 第一步：创建职位

1. 进入「职位管理」
2. 点击 **添加职位**
3. 填写：职位名称（必填）、描述、薪资范围、招聘人数、工作地点
4. 点击 **保存**

### 第二步：添加候选人

有两种方式：

**方式一：手动添加**
1. 进入「候选人」页面
2. 点击 **添加候选人**
3. 填写：姓名（必填）、电话（必填）、邮箱、应聘职位
4. 提交后自动弹出「启动面试流程」

**方式二：PDF 简历导入**
1. 点击 **导入候选人**
2. 选择 PDF 简历文件（支持 BOSS 直聘导出格式）
3. 系统自动解析姓名、电话、邮箱
4. 确认信息后点击 **确认导入**

### 第三步：启动面试流程

1. 在候选人列表中点击 **启动面试流程**
2. 选择应聘职位
3. 配置面试轮次（默认3轮）：

   | 轮次 | 类型 | 说明 |
   |------|------|------|
   | 第1轮 | HR 初面 | 面试官默认为当前 HR |
   | 第2轮 | 技术面试 | 选择技术面试官 |
   | 第3轮 | 终面 | 选择终面面试官 |

4. 可点击 **+ 添加轮次** 增加技术面（最多4轮）
5. 终面始终在最后，不可删除
6. 点击 **启动流程**

### 第四步：安排面试

1. 进入面试流程详情页
2. 在对应轮次点击 **安排面试**
3. 填写：
   - 开始时间、结束时间（必填）
   - 面试形式：线上 / 线下
   - 线上填：会议链接、会议号
   - 线下填：面试地点
4. 可勾选「通知候选人（发送邮件）」
5. 点击 **确认**
6. 系统自动发送邮件通知面试官（如有邮箱）

### 第五步：面试官填写反馈

**方式一：登录后填写**
1. 面试官登录系统
2. 进入「我的面试」
3. 点击对应面试的 **填写反馈**
4. 填写：总体评价（通过/不通过/待定）、评分（1-5星）、优势、待提升项、备注
5. 提交

**方式二：邮件链接（免登录）**
- 面试官收到邮件，点击链接直接填写反馈
- 无需登录系统

### 第六步：HR 推进流程

当一轮面试完成后，流程进入「等待 HR 确认」状态：

1. 进入面试流程详情
2. 看到提示「第X轮已完成，请HR确认下一步」
3. 选择操作：
   - **通过，安排下一轮** → 流程进入下一轮
   - **直接录用** → 候选人状态变为「已录用」
   - **不通过** → 候选人状态变为「已拒绝」

## 用户管理

### 添加面试官

1. 进入「用户管理」
2. 点击 **添加用户**
3. 填写：用户名、密码、姓名、邮箱（选填）、角色
4. 点击 **添加用户**

### 停用/启用面试官

- 停用：点击面试官行的 **停用** 按钮（历史数据不受影响）
- 启用：点击已停用面试官的 **启用** 按钮
- 已停用用户显示灰色 + "已停用"标签
- HR 账户不可被停用

## 邮件通知

系统会在以下场景自动发送邮件（发件人：hrgroup@malong.com）：

| 场景 | 收件人 | 内容 |
|------|--------|------|
| 安排面试 | 面试官 | 面试时间、候选人、形式、会议链接 |
| 安排面试（勾选通知） | 候选人 | 面试邀请通知 |
| 轮次完成 | HR | 本轮结果，提醒确认下一步 |
| 流程完成 | HR | 最终结果（录用/拒绝） |
| @面试官 | 被@面试官 | 查看候选人简历提醒 |
| 反馈提醒 | 面试官 | 填写反馈链接 |

## 站内通知

- 页面右上角铃铛图标，红色数字显示未读通知数
- 点击查看通知列表
- 支持单条标记已读或全部标记已读
- 通知包含跳转链接，点击可直达相关页面

## 候选人状态说明

| 状态 | 含义 |
|------|------|
| 待处理 | 新添加，未开始面试 |
| 筛选中 | 简历筛选阶段 |
| 初试 | 第一轮面试进行中 |
| 复试 | 技术面试进行中 |
| 终试 | 终面进行中 |
| 已录用 | 面试通过，已录用 |
| 已拒绝 | 面试未通过 |

状态随面试流程自动更新，HR 也可通过流程操作手动推进。

## 常见问题

**Q: 面试官看不到面试安排？**
A: 面试官只能看到分配给自己的面试，确保在流程配置中选择了正确的面试官。

**Q: 简历导入失败？**
A: 目前支持 PDF 格式，推荐使用 BOSS 直聘导出的简历。文件大小限制 20MB。

**Q: 停用面试官后历史数据还在吗？**
A: 在的。停用只是禁止该账户登录和被分配新面试，所有历史面试记录和反馈完整保留。

**Q: 邮件没有收到？**
A: 确认面试官/候选人的邮箱地址填写正确。飞书邮箱限制：200封/100秒，单日上限6000封。

**Q: 最多支持几轮面试？**
A: 最多4轮（HR初面 + 最多2轮技术面 + 终面）。终面始终是最后一轮。
