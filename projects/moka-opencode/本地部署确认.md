## 🔄 常用命令

### 查看服务状态

```bash
# 查看后端进程
ps aux | grep "nest" | grep -v grep

# 查看前端进程
ps aux | grep "next" | grep -v grep

# 查看端口占用
lsof -i :3000
lsof -i :3001
```

### 重启服务

```bash
# 如果服务异常，可以重启

# 1. 停止现有进程
kill 3501803  # 后端PID
kill 2497428  # 前端PID

# 2. 重新启动后端
cd /home/malong/projects/moka-opencode/backend
bun run start:dev

# 3. 重新启动前端
cd /home/malong/projects/moka-opencode/frontend
bun run dev
```

### 查看日志

```bash
# 查看后端日志（如果使用了日志文件）
tail -f /home/malong/projects/moka-opencode/backend/logs/app.log

# 或者直接在终端查看输出
```
