# 会话配置

## 语言偏好

- **主要语言**: 中文 (Chinese)
- 所有响应、解释和文档都应该使用中文
- 代码注释和文档可以保持英文（如果项目约定如此）

## 项目要求

- **端口配置**:
  - 前端: 13000
  - 后端: 13001
- **技术栈**: NestJS + Next.js + Prisma + Docker
- **部署环境**: Ubuntu 20.04+ with Docker Compose

## 沟通风格

- 简洁直接，避免冗余的客套话
- 技术细节要准确完整
- 问题解决要提供具体可执行的步骤
- 优先使用中文进行所有交流

## 工作流程

- 先确认理解需求
- 提供解决方案或执行计划
- 执行后验证结果
- 记录关键决策和配置变更
